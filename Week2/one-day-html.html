Learn HTML in 1 Day — Quick Beginner Guide

This page is a compact, practical plan to learn the basics of HTML in one day. Read, copy the examples, and edit them in your own files.

Plan (one day)

  1. Intro: What is HTML? — 15 minutes
  2. Structure: Tags, elements, attributes — 45 minutes
  3. Content: text, lists, images, links — 60 minutes
  4. Layout & forms: simple form fields and semantic tags — 45 minutes
  5. Practice exercises & tweak styles — 45 minutes

1 — Basic page structure

Every page starts with a few standard tags. Try creating a new file called index.html and paste this:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width,initial-scale=1" />
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello world!</h1>
    <p>I just wrote my first HTML file.</p>
  </body>
</html>
Exercise: Save that as index.html and open it in your browser. Change the text inside <h1> and refresh.

2 — Common elements you should know

  • <h1> ... </h1> — headings (h1 highest, h6 smallest)
  • <p> ... </p> — paragraphs
  • <a href="...">link</a> — links
  • <img src="..." alt="..." /> — images (always add alt)
  • <ul> <li> ... </li> </ul> — bullet lists
<p>Visit <a href="https://example.com">example.com</a>.</p>
<img src="https://via.placeholder.com/200" alt="placeholder image" />
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>
Exercise: Add an image and a link to your index.html. Try changing the link target to open in a new tab using target="_blank".

3 — Forms (very simple)

Forms let users type input. This small example shows a text field, email and a submit button:

<form>
  <label for="name">Name</label>
  <input id="name" name="name" type="text" placeholder="Your name" />

  <label for="email">Email</label>
  <input id="email" name="email" type="email" placeholder="you@example.com" />

  <button type="submit">Send</button>
</form>
Exercise: Try adding the form to your page. Submit will do nothing locally (unless you set an action) — that’s fine for practicing.

4 — Semantic tags you should learn next

These tags make your HTML clearer and help search engines and screen readers:

  • <header>, <nav>, <main>, <footer>
  • <section>, <article> — for meaningful pieces of content

5 — Quick practice plan (1 day)

  1. 0–30 min: Create and open a basic page. Edit text & tags — get comfortable.
  2. 30–90 min: Add images, links and lists. Practice copying examples.
  3. 90–120 min: Build a small contact form and practice semantic tags.
  4. 120–240 min: Make a small page with 3 sections and style with inline CSS (later learn external CSS).

Extra — Tiny starter you can copy (editable)

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width,initial-scale=1" />
    <title>My Practice Page</title>
    <style> body{font-family:Arial;padding:18px} header{border-bottom:1px solid #eee;padding-bottom:10px;margin-bottom:10px} .grid{display:grid;grid-template-columns:1fr 1fr;gap:10px}</style>
  </head>
  <body>
    <header>
      <h1>My Practice Page</h1>
      <p>A tiny page I made while learning HTML</p>
    </header>

    <main>
      <section class="grid">
        <div><h2>About</h2><p>I am learning HTML today.</p></div>
        <div><h2>Contact</h2><p>Email: <a href="mailto:you@example.com">you@example.com</a></p></div>
      </section>
    </main>

    <footer><p>Made by you — practice makes progress!</p></footer>
  </body>
</html>

  • Use headings, paragraphs, and lists — done
  • Add images and links — done
  • Create a simple form — done
  • Build a small multi-section page — done
